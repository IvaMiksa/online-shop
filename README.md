## Installation & Setup

### **Option 1: Run with Docker (Recommended)**
#### Prerequisites:
- Docker Desktop installed locally

#### **Start the App**:
docker compose up


### **Option 2: Install manually**
- Install dependencies:
npm install

- Run dev server:
npm run dev
